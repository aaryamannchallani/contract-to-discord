# contract-to-discord
Integrate your contracts with discord webhooks!

## Environment Variables
<ul>
  <li>DISCORD_WEBHOOK_URL</li>
  <li>WEBHOOK_NAME</li>
  <li>INFURA_URL</li> (note:use ws)
  <li>CONTRACT_ABI</li>
  <li>CONTRACT_ADDRESS</li>
 </ul>
  
Use `yarn install` to install the dependencies
